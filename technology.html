System Architecture

LogiGuard360 follows a modern three-tier architecture pattern, separating concerns between presentation, business logic, and data persistence for maintainability and scalability.

Presentation Layer

Browser-based frontend using A-Frame and Three.js for 360¬∞ VR environments. Responsive UI built with HTML5/CSS3 adapts to desktop, mobile, and VR headsets.

Application Layer

Node.js/Express.js backend handles API requests, user authentication, quiz validation, and progress tracking. RESTful endpoints serve training content and analytics.

Data Layer

MongoDB stores user profiles, completed modules, quiz scores, threat scenarios, and training analytics. Flexible schema supports rapid feature development.

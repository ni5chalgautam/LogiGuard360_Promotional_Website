Welcome to LogiGuard360

LogiGuard360 is a cybersecurity training platform for logistics staff, built around a 360° virtual warehouse environment, interactive hotspots, and scenario-driven learning to improve awareness and safer decision-making.

Project Overview Explore Features

360° Virtual Warehouse Environment

Immersive 3D Training Experience

Step into photorealistic virtual warehouse environments designed to simulate real-world logistics settings where spatial awareness and context matter.

What Makes It Special

Realistic warehouse scenarios
Clickable learning hotspots
Contextual threat identification
Short, focused learning bursts

Threat Hotspots

Identify Security Risks in Context

Discover security vulnerabilities by interacting with hotspots placed throughout the warehouse environment. Learn safe behaviours in the exact context where threats occur.

Example Hotspots

Phishing emails & suspicious links
Unknown USB devices
Unlocked terminals
QR code scams

Scenario-Based Quizzes

Test What You Learn

Apply knowledge through realistic scenarios and quizzes. These activities help reinforce correct responses and identify areas needing reinforcement.

Learning Outcomes

Recognise phishing indicators
Practice safe device behaviour
Report suspicious activity
Respond confidently to threats
